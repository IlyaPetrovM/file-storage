# File Storage Service

REST API сервис для управления файлами с поддержкой загрузки, скачивания и удаления файлов.

## Быстрый старт

### Docker
```bash
docker-compose up --build
```

Сервис запустится на `http://localhost:3001`

### Локально
```bash
npm install
npm start
```

## API Endpoints

### Информация о сервисе
- **GET** `/` - Основная информация и список доступных endpoints
- **GET** `/health` - Проверка здоровья сервиса

### Работа с файлами

#### Загрузить файл
```
POST /api/files
Content-Type: multipart/form-data

file: <binary>
```
**Ответ (201):**
```json
{
  "message": "File uploaded successfully",
  "file": {
    "id": "1777218058633-57656008-filename.ext",
    "originalName": "filename.ext",
    "size": 1024,
    "mimeType": "image/jpeg",
    "path": "/app/storage/1777218058633-57656008-filename.ext",
    "uploadedAt": "2026-05-15T10:30:00.000Z"
  }
}
```

#### Список всех файлов
```
GET /api/files
```
**Ответ (200):**
```json
{
  "count": 2,
  "files": [
    {
      "id": "1777218058633-57656008-image.jpg",
      "size": 102400,
      "createdAt": "2026-05-15T10:30:00.000Z",
      "modifiedAt": "2026-05-15T10:30:00.000Z"
    }
  ]
}
```

#### Скачать файл
```
GET /api/files/:id
```
**Параметры:**
- `id` - ID файла (полное имя файла)

**Ответ (200):** Бинарное содержимое файла

#### Удалить файл
```
DELETE /api/files/:id
```
**Параметры:**
- `id` - ID файла (полное имя файла)

**Ответ (200):**
```json
{
  "message": "File deleted successfully",
  "id": "1777218058633-57656008-filename.ext"
}
```

## Конфигурация

Переменные окружения (установлены в docker-compose.yml):

| Переменная | Значение по умолчанию | Описание |
|---|---|---|
| `PORT` | 3001 | Порт сервиса |
| `NODE_ENV` | development | Окружение (development/production) |
| `STORAGE_PATH` | ./storage | Путь к директории хранения файлов |
| `MAX_FILE_SIZE` | 52428800 (50MB) | Максимальный размер загружаемого файла |

## Особенности

- **Автоматическое создание имён** - Файлы получают уникальные имена для избежания конфликтов
- **Валидация размера** - Максимальный размер файла: 50MB (можно изменить через переменную окружения)
- **Логирование** - Все операции логируются
- **CORS** - Включена поддержка cross-origin запросов
- **Хранилище на диске** - Файлы сохраняются в локальной директории
